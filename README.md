Gym Membership Management System

A backend application built with Spring Boot and Spring Data JPA that models a basic gym membership system.
This project focuses on clean architecture, database modeling, and backend fundamentals using Java.

Project Overview

This application represents a simple gym management backend, designed to handle gym members and their memberships.
It demonstrates how a real-world domain can be modeled using JPA entities and repositories in a Spring Boot application.

The project is intended as a learning and portfolio project, showcasing core backend development skills.
